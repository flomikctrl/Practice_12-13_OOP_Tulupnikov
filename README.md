## Описание проекта
Консольное приложение для управления библиотекой, демонстрирующее основные принципы объектно-ориентированного программирования.

## Реализованная функциональность
Основные возможности:
- Добавление книги в библиотеку
- Выдача и возврат книги
- Поиск книг по автору
- Просмотр доступных книг
- Сохранение и загрузка данных библиотеки

## ООП-компоненты
1. **Book (Модель)**
   - Свойства: `Id`, `Title`, `Author`, `Year`, `Genre`, `IsAvailable`
   - Методы: `ToString()`

2. **Library (Модель)**
   - Управление коллекцией книг
   - Методы поиска и фильтрации
   - Инкапсуляция данных

3. **LibraryService (Сервис)**
   - Бизнес-логика работы с библиотекой
   - Обработка операций добавления, изменения, удаления и поиска
   - Сохранение и загрузка данных

## Как запустить
Требования:
- .NET SDK 6.0 или выше

Запуск:
```bash
cd src/LibraryApp
dotnet run
```

## Применённые принципы ООП
1. **Инкапсуляция**
   - Приватные поля и свойства в классах
   - Управление коллекцией книг через методы класса `Library`

2. **Абстракция**
   - Разделение модели и бизнес-логики
   - Простой интерфейс для пользователя через `LibraryService`

3. **Модульность**
   - Разделение класса `Book`, `Library` и `LibraryService` по файлам
   - Организация кода в пространстве имён `LibraryApp`
